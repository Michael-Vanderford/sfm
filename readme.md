# SFM

Simple File Manager (SFM) is a lightweight and efficient file management tool designed for easy navigation, organization, and manipulation of files and directories.

## Features

- Browse files and directories
- Create, rename, and delete files/folders
- Move and copy files
- Search functionality
- User-friendly interface

## Installation


- git clone https://github.com/yourusername/sfm.git
- cd sfm
- npm install --save-dev
- npm start

# Build instructions

- npm run build
